# MinkNightmareDriver
Mink Nightmare Driver (replaces [JsonWireProtocol](https://github.com/SeleniumHQ/selenium/wiki/JsonWireProtocol) and [PhantomJS](http://phantomjs.org/) with [Nightmare](http://www.nightmarejs.org/))

Nightmare via electron is between 2 to 3 times faster than PhantomJS ([source](https://github.com/segmentio/nightmare/issues/484)).

## Requirements
- PHP and of course composer
- Node.js and nightmare (via npm)
